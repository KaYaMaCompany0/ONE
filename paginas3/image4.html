Sensores de Arduino

¡Descubre nuestra variedad de sensores para Arduino! Perfectos para tus proyectos de electrónica y automatización(sensor Ultrasónico como referencia).

Opciones:

- Sensor de Temperatura y Humedad DHT11:

Ideal para medir la temperatura y humedad en interiores y exteriores.

- Sensor Ultrasonido HC-SR04:

Utilizado para medir distancias, ideal para proyectos de robótica y control de obstáculos.

- Sensor de Movimiento PIR:

Detecta movimiento a través de cambios en la radiación infrarroja.

- Sensor de Luz LDR:

Mide la intensidad de la luz, ideal para proyectos de automatización de iluminación.

- Sensor de Gas MQ-2:

Detecta gases como LPG, humo y otros gases inflamables.

- Sensor de Presión BMP180:

Mide la presión atmosférica y la temperatura, ideal para proyectos meteorológicos.

- Sensor de Proximidad IR:

Detecta objetos cercanos sin contacto físico, ideal para aplicaciones de automatización.

- Sensor de Vibración SW-420:

Detecta vibraciones y movimientos, ideal para alarmas y monitoreo.

- Sensor de Agua:

Detecta la presencia de agua, ideal para sistemas de riego automatizados.

- Sensor de Toque TTP223:

Detecta toques en una superficie, ideal para interfaces de usuario interactivas.
