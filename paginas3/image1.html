Arduino UNO Genérico

¡Descubre nuestra amplia gama de módulos Arduino! Perfectos para tus proyectos de electrónica y programación.

Opciones:

- Arduino UNO Genérico:

Una excelente opción para principiantes, ofrece todas las funcionalidades necesarias para iniciar tus proyectos.

- Arduino UNO Original:

La versión oficial, confiable y de alta calidad, ideal para proyectos más serios.

- Arduino Nano:

Compacto y versátil, perfecto para proyectos donde el espacio es limitado.

- Arduino Mega:

Con más pines y mayor memoria, ideal para proyectos complejos que requieren más recursos.

- Arduino Leonardo:

Permite emular un teclado o un mouse, perfecto para proyectos interactivos.

- Arduino Yun:

Combina la potencia de Arduino con la conectividad de Linux, ideal para proyectos IoT.
